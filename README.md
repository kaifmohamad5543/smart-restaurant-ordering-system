# smart-restaurant-ordering-system
Smart Restaurant Ordering System is an advanced Java-based desktop application developed using Object-Oriented Programming principles, MVC architecture, Java Swing GUI, and multiple software design patterns. The system allows restaurant staff and customers to manage food ordering, online delivery orders, customer registration, menu management, sales analytics, notifications, receipt printing, and daily sales tracking through an interactive graphical dashboard.

The project implements advanced software-engineering concepts including Factory Pattern, Strategy Pattern, Singleton Pattern, encapsulation, abstraction, polymorphism, password hashing, security validation, and JUnit 5 testing. The application also provides dynamic restaurant sales graphs, online order management, add/delete menu-item functionality, print-ready receipts, and real-time notification handling.
